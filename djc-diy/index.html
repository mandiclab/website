THE WORLD'S FIRST DIY DJ CONTROLLER

DJC-DIY

Difficulty: Normal

Print Time: 12h

Assembly Time: 30min

This is basic project information.

  • Size: 300x200x50mm
  • Power: USB-C
  • Output: MIDI via USB
  • Arduino Pro Micro
  • Rotary Encoders x2
  • Potentiometers x6
  • Push Buttons x12

To use the controller, plug it via USB and map it inside your DJ software (Mixxx, Traktor, etc).

  • Q: Does it work on Mac?
    A: Yes, it's MIDI class-compliant.
  • Q: Can I use other software?
    A: Yes, any that supports MIDI mapping.
