DJC-DIY

Difficulty: Normal

Print Time: 12h

DJC-DIY is the world's first open-source DIY DJ controller you can build yourself. No custom PCB, no special tools, no advanced skills. Just some time and patience.

Deck A, Deck B, Mixer

    Deck
  • Jog Wheel - Only for pitch bend, no scratching for now.
  • Play/Pause
  • Cue
  • 2x Performance Pads
    Mixer
  • Low EQ
  • FX/EQ/Volume
  • Crossfader
  • 1 x Arduino Pro Micro - ATmega32U4 5V
  • 8 x Push Button - B3F-4055 - 12x12x7.3mm
  • 4 x Rotary Potentiometer - B103 - RV09 - D Shaft - 23mm
  • 3 x Slider Potentiometer - B103 - 60mm -
  • 2 x Rotary Encoder - EC11 - D Shaft - 15mm - No push button recommended
  • 24AWG Wire - Red, Black, Blue, Green, Yellow

Q: Does it work with Virtual DJ, Traktor, or Rekordbox?

A: It currently works out of the box only with Mixxx, but the controller sends standard MIDI messages, so it can be manually mapped to other DJ software.


Q: Is it for sale?

A: For now, we’re not selling the device or any DIY kits.


Q: How much does it cost?

A: If you have access to a 3D printer (or a friend who can print for you), the whole build can cost as little as $15 — plastic included.


Q: Can I add more features (buttons, lights, screen...)?

A: This is just the first version. We’ll try to expand it to the hardware limits in future iterations — and you're welcome to modify it too!
